# Forma

Adaptive personal dashboard that creates customized modules based on user goals, priorities, and tracking needs.

## Version 1

Version 1 contains the onboarding experience and a generated personal operating system. The user selects all identities that fit them, then describes outcome, friction, time horizon, and structure preference; Forma combines those signals to infer modules, primary system, navigation, starter content, and recommendations. Finance includes a full local MoneyMap-style module with monthly cycles, currency selection, transactions, budgets, savings goals, investments, analytics, demo data, and rule-based insights.

Open `index.html` in a browser to run it locally. Onboarding data, module data, personalized recommendations, and the generated workspace configuration are stored in `localStorage`. Configuration export lives in Settings.
